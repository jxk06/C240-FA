# C240-FA

A responsive frontend prototype website for the "Label Detectives" allergy-safe grocery scanning solution.

## Website files

- `index.html` — responsive landing page with hero section, solution details, audience messaging, how it works, and demo simulation.
- `styles.css` — mobile-first styling and polished UI for desktop and mobile.
- `script.js` — interactive navigation toggle and simulated scan demo.

## How to view

Install dependencies and start the site locally:

```bash
npm install
npm run start
```

Then open `http://localhost:3000` in your browser.
